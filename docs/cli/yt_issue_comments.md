## yt issue comments

List comments on an issue

### Synopsis

List all comments on a YouTrack issue, showing author, timestamp,
and text for each comment.

```
yt issue comments <id> [flags]
```

### Examples

```
  # list comments
  yt issue comments PROJ-123

  # JSON output
  yt issue comments PROJ-123 --json
```

### Options

```
  -h, --help   help for comments
```

### Options inherited from parent commands
