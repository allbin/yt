## yt link types

List available issue link types

### Synopsis

List the instance's issue link types and their directed phrases.

Use the listed phrases (in kebab, spaced, or squashed form) as the relation
argument to "yt link" and "yt unlink".

```
yt link types [flags]
```

### Examples

```
  # list link types
  yt link types

  # JSON output
  yt link types --json
```
