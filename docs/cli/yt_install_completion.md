## yt install completion

Install shell completions

### Synopsis

Install shell completion script for tab-completion of yt commands and flags.

Auto-detects shell from $SHELL. Supported: bash, zsh, fish.
Use --shell to override detection.

```
yt install completion [flags]
```

### Examples

```
  yt install completion
  yt install completion --shell fish
```

### Options

```
  -h, --help           help for completion
      --shell string   override shell detection (bash|zsh|fish)
```
